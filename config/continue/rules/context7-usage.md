---
name: Context7 Usage Rules
description: Rules for interacting with Context7 MCP server to look up relevant documentation or code snippet.
---

1. Prefer Context7 over any web search tools when looking up for documentation or code snippet.
2. If Context7 returns no relevant results after 3 attempts, then use a web search tool.
3. When using `get-library-docs` tool, set `tokens: 5000` in the arguments.
