---
name: Context7 Usage Rules
description: Rules for interacting with Context7 MCP server to look up relevant documentation or code snippet.
---

1. Prefer Context7 over any Web search tool when looking up for documentation or code snippet.
   Only use a Web search tool when the relevant documentation or code snippet is not available on Context7.
2. `get-library-docs` Tool
   - Always set `tokens: 5000` in the arguments.
